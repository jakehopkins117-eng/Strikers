STRIKERS v3.5 — SPORTSBOOK INTELLIGENCE

1. Create a new API key in The Odds API dashboard because the previous key was shared in chat.
2. In the Strikers2 folder, copy .env.example to .env.
3. Replace ODDS_API_KEY=replace_with_your_new_key with the new key.
4. Run: pip install -r requirements.txt
5. Restart the backend and frontend.
